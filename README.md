# AI Resume Builder

AI Resume Builder is a simple tool that helps users create professional resumes using AI.

## Features
- AI-generated resumes
- Simple user interface
- Fast resume creation
- Mobile friendly

## Problem
Many people struggle to create professional resumes quickly.

## Solution
This project helps users generate resumes easily using AI prompts and templates.

- HTML

- JavaScript
- AI API
